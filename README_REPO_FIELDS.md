# Recommended GitHub repository creation fields

## Repository name

```text
local-ai-chat-exporter
```

Alternatives:

```text
chatvault-local-exporter
ai-chat-local-exporter
open-chat-exporter
```

Best choice: `local-ai-chat-exporter` because it avoids implying official ChatGPT/OpenAI affiliation and leaves room for Claude/Gemini/Perplexity later.

## Description

```text
Open-source browser extension to export AI chats locally to Markdown, PDF, JSON, CSV, TXT, HTML and DOCX. No telemetry.
```

## Visibility

Public.

Enable README or add this pack immediately after creating the repo.

Choose Node.

## License

Recommended: GPL-3.0-or-later if you want strong open-source/copyleft positioning.

Alternative: MPL-2.0 if you want easier business adoption while keeping file-level copyleft.

Alternative: MIT if you want maximum reuse and minimum friction, but competitors can more easily reuse your code in closed products.
