# Privacy Policy

Local AI Chat Exporter processes conversation content locally in your browser only after you explicitly click an export action.

## What data is processed

The extension reads the visible AI chat conversation in the active browser tab in order to generate local export files.

## What data is collected

None by default.

## What data is transmitted

None in v1. The extension does not send conversation content to any external server.

## Analytics and telemetry

No analytics. No telemetry. No ads. No trackers. No remote logging.

## Storage

The extension may store local preferences such as selected export format or filename template. It does not store full conversation content by default.

## Optional integrations

Future optional integrations, if added, must be disabled by default and must require explicit user action before sending selected exports to a user-chosen destination.

## Contact

Open an issue in the GitHub repository for privacy questions or concerns.
