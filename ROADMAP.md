# AI Chat Export Roadmap

## Business Model

- Core free/open-source: current-tab export, local rendering, local settings, and supported format
  improvements remain free in v1.
- Donations: GitHub Sponsors and OpenCollective support maintenance, docs, provider fixtures, and
  compatibility updates.
- Paid support: optional setup help, QA guidance, and migration support.
- Custom enterprise build: optional packaging or policy-specific variants for organizations.
- Optional future cloud companion: may be explored later, but not core and not required for local
  exports.

## Near-Term Priorities

1. Provider QA depth for stable public claims.
2. Export fidelity for code, tables, citations, and attached visible context.
3. Accessibility and compact popup polish.
4. Documentation and store listing readiness.
5. Reproducible release QA with no generated artifact drift.

## Non-Goals For Core V1

- Ads or telemetry.
- Pricing gates for core export formats.
- Server-side export rendering.
- Account-wide background scraping.
- Default transcript persistence.
