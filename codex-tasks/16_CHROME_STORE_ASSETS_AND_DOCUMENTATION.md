# Task 16 — Chrome Web Store assets and submission documentation

## Goal

Prepare the public listing package: descriptions, screenshots checklist, permission copy, and reviewer instructions.

## Files to create/update

```text
docs/store-listing.md
docs/reviewer-instructions.md
docs/screenshots-checklist.md
docs/permissions.md
PRIVACY.md
README.md
```

## Requirements

1. Store listing must include:

- title;
- short description;
- detailed description;
- features;
- privacy claims;
- supported browsers;
- supported platforms;
- disclaimer of non-affiliation.

2. Permission justifications must match manifest.
3. Reviewer instructions must be simple:

```text
Open ChatGPT, open any conversation, click extension, scan, export Markdown.
```

4. Screenshots checklist:

- popup idle;
- scan progress;
- completeness report;
- export options;
- preview;
- downloaded Markdown example;
- privacy screen.

5. README must include:

- install from source;
- local development;
- privacy model;
- supported formats;
- limitations;
- troubleshooting.

## Acceptance criteria

- No misleading affiliation language.
- No unsupported feature claims.
- Privacy policy matches code.
- Docs include Vivaldi local install instructions.
- `pnpm check` passes.

## Deliverable

Commit title:

```text
Prepare Chrome Web Store listing documentation
```
