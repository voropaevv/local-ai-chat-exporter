# Chrome Web Store Listing Draft

## Name

LogThread

## Short Description

Export AI chat threads locally. No account, no telemetry, no server upload.

## Description

LogThread is a free, open-source, local-first browser extension for exporting AI chat conversations to local files.

Supported outputs include Markdown, TXT, JSON, CSV, HTML, DOCX, print-ready PDF HTML, PNG snapshots, and ZIP bundles. ChatGPT is the primary supported platform. Claude, Gemini, Perplexity, and NotebookLM are visible-message adapters and should be verified before relying on an export.

LogThread does not include telemetry, analytics, ads, trackers, remote logging, remote rendering, or external export servers. Export actions are initiated by the user and run locally in the browser extension.

## Privacy Claims

- Local processing after explicit user action.
- No server upload.
- No telemetry or analytics.
- No account required.
- Minimal Manifest V3 permissions.

## Do Not Claim

- Do not claim account-wide history export.
- Do not claim full secondary-provider support until live QA passes.
- Do not claim medical, legal, financial, or compliance-grade archival guarantees.
