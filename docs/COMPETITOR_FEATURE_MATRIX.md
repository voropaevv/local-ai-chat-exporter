# Competitor Feature Matrix

Use this as a target feature checklist, not as copied implementation.

## Common competitor features

- Export to Markdown.
- Export to PDF.
- Export to TXT.
- Export to JSON.
- Export to CSV.
- Export to HTML.
- Export to DOCX/Word.
- Copy full conversation.
- Partial export / selected messages.
- Code block support.
- Table support.
- Math/LaTeX preservation.
- Image export or references.
- Filename customization.
- Obsidian export.
- Notion/Drive export.
- Batch export.
- Multiple AI platforms.
- Open-source claim.
- Local-only claim.

## Gaps to dominate

- Verifiable no-telemetry build.
- Minimal permissions, no `all_urls`.
- Completeness report for long chats.
- Reproducible release ZIP hash.
- UI selector fallback tests.
- Fixture-based regression tests after ChatGPT UI changes.
- Vivaldi-specific local testing docs.
- Redaction before export.
- Export schema stability.
- Obsidian/GitBook/GitHub Markdown profiles.
- Optional integrations separated from local core.

## Strategic rule

Do not compete by adding risky features first. Compete by being the safest and most reliable exporter. Features that require cloud accounts or broad permissions must remain optional.
