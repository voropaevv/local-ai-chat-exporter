Privacy model

No telemetry, no remote rendering, no export server.

The extension processes the current tab locally after an explicit user action. Browser storage is used for preferences and only stores conversation content if you opt into the Local Library.

Long chat support

Scroll collection and completeness checks for long threads.

AI Chat Export tracks first and last messages, duplicate counts, scan warnings, selected ranges, and platform-specific limitations so exports do not silently overclaim coverage.

Research exports

Preserve citations, code, tables, reasoning settings, and metadata.

Advanced ChatGPT extraction keeps rich content structured where fixtures support it, while redaction settings help prepare safer files for review.

Obsidian / Markdown

Profiles for default archives, Obsidian, GitHub, GitBook, and research logs.

Filename templates and frontmatter options make exports predictable without turning the popup into a cloud workspace.

Batch export

Export already-open AI chat tabs into one ZIP.

Batch export remains user-initiated, requests optional permissions only when needed, and includes manifest warnings when a selected tab fails.

FAQ

Common questions before installing.

Does AI Chat Export upload chats?

No. Export rendering runs locally in the extension.

Is there a pricing wall?

No pricing wall in v1. Donation and support links are optional.

Does it export account history?

No. It exports the current supported chat tab after you click.

GitHub / Sponsor

Open-source project with optional support.

Review the source, build locally, and sponsor ongoing maintenance if the tool saves you time. Core export features stay free in v1.

Business model

Donation-supported local tooling without a pricing wall.

  • Core exports stay free and open-source.
  • Donations fund maintenance, docs, provider fixtures, and compatibility updates.
  • Paid support and custom enterprise builds are optional services.
  • Any future cloud companion must stay optional and separate from core local exports.

Privacy

Local processing after explicit user action.

No account, telemetry, analytics, ads, trackers, session replay, remote logging, remote code, or export server.

Security

Minimal Manifest V3 permissions.

AI Chat Export avoids broad hosts and sensitive permissions. Optional downloads, tabs, and host access are requested only for user-facing workflows.
